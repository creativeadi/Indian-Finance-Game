# 🎲 Indian Finance Game

A Unity-based strategy board game inspired by Monopoly, enhanced with a **probability-driven dynamic pricing system** that simulates real-world financial behavior and investment decision-making.

---

## 📌 Overview

Indian Finance Game is not just a traditional board game clone—it introduces a **data-driven economic model** where property prices fluctuate based on predefined probabilities. Players must analyze trends, manage capital, and make strategic investment decisions.

This project combines:
- Game development (Unity)
- Economic modeling
- Probability-based logic
- UI/UX interaction systems

---

## 🚀 Core Features

### 🎯 Dynamic Property Pricing
- Each city has an associated probability value
- Prices increase/decrease based on probability changes
- Mimics demand-supply economics

### 🎲 Dice Roll System
- Randomized player movement
- Turn-based gameplay logic

### 💰 Player Economy System
- Real-time money updates
- Property buying mechanics
- Balance tracking using UI

### 🏙️ Custom Game Board
- Indian and global cities
- Expandable tile system

### 🧠 Strategy-Oriented Gameplay
- Players must evaluate:
  - Risk vs reward
  - Property value trends
  - Long-term gains

---

## 🧮 Probability-Based Pricing Model

Each property is assigned a probability score:

- Higher probability → Higher price growth
- Lower probability → Slower or negative growth
